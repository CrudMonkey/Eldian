## Synopsis

Eldian is a password management software that helps in organizing your online/offline accounts. This software also includes a password analyzer and a password generator. Also includes an added feature for storing notes (text files) which has .ldn extension that is secure and encrypted and can be opened only by this software. Also has a guest mode with limited functionality of creating secure notes.
